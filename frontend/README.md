AST-Frontend
## Getting Started

First, run the development server:

```bash
yarn install
yarn dev
```

# Homepage 
* `http://127.0.0.1:3000`
* ast tree without backend api

# AST Backend page 
* `http://127.0.0.1:3000/ast-backend`
* ast tree with backend api (`https://127.0.0.1:5001`)
* run backend api.
